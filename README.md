# 👁️ Nuni: AI 음성 비서 기반 시각장애인 전용 내비게이션

**Nuni(누니)**는 시각장애인의 안전하고 독립적인 보행을 지원하기 위해 개발된 **AI 음성 비서 기반 전용 내비게이션 서비스**입니다.
점자블록 중심의 경로 안내와 실시간 장애물 탐지, 그리고 위급 상황 관제 시스템을 하나로 통합했습니다.

---

## 📌 1. 프로젝트 개요 (Overview)

### **배경 및 필요성**
* **장애인 실태조사(2023):** 시각장애인이 외출 시 겪는 가장 큰 불편은 '편의시설 부족(56.6%)'과 '동반자 부재(33.6%)'로 나타났습니다.
* **기존 내비게이션의 한계:** 점자블록 중심의 정밀한 경로 안내가 부족하며, 이동 경로상의 실시간 장애물(불법 주차, 킥보드 등)에 대한 대응이 어렵습니다.
* **해결 방안:** 시각장애인의 보행 특성을 고려한 '점자블록 기반 최적 경로'를 제공하고, 온디바이스 AI로 지연 시간 없이 장애물을 감지하여 알립니다.

---

## 🚀 2. 핵심 기능 (Key Features)

### **1) AI 음성 비서 "누니"**
* **의도 분류:** 로컬 LLM(**Mistral Nemo Instruct 2407**)을 연동하여 사용자의 음성 명령에서 목적지나 도움 요청 의도를 빠르게 추출합니다.
* **빠른 응답:** 복잡한 추론보다 빠른 분류에 최적화된 모델을 선정하여 평균 호출 시간을 단축했습니다.

### **2) 정밀 보행 및 장애물 탐지**
* **경로 안내:** 점자블록 데이터 기반으로 최적의 보행 경로를 생성하고, GPS를 통해 사용자의 위치를 능동적으로 보정하며 TTS로 안내합니다.
* **실시간 객체 탐지:** **YOLO26n** 경량화 모델을 사용하여 온디바이스에서 점자블록 위 장애물(자전거, 킥보드, 오토바이 등)을 실시간 감지합니다.
* **위험 경고:** 점자블록과 장애물의 겹침(IoU)을 판단하여 위험 시 즉각적인 음성 경고를 출력합니다.

### **3) 통합 관제 시스템**
* **실시간 모니터링:** 관제 웹에서 사용자의 현재 위치, 이동 경로, 카메라 영상 스트리밍을 실시간으로 확인합니다.
* **긴급 대응:** 위급 상황 발생 시 관제 센터에 즉각 알림이 표시되며, 관제사가 119 출동이나 센터 지원을 연결할 수 있습니다.

---

## 🛠️ 3. 기술 스택 (Tech Stack)

### **Client & AI**
* **Android:** Java, CameraX (영상 수집), TFLite (온디바이스 추론)
* **Object Detection:** Ultralytics YOLO (YOLO26n 최적화)

### **Server & LLM**
* **Backend:** Python, FastAPI
* **LLM Engine:** LM Studio (Mistral Nemo Instruct 2407)

### **Web & Visualization**
* **Frontend:** React, Vite 
* **Maps:** OpenStreetMap, Turf (지리 데이터 처리), T-Map API 

---

## 🏗️ 4. 시스템 아키텍처 (System Architecture)
Android 앱에서 수집된 음성 및 영상 데이터가 FastAPI 백엔드로 전송되며, LLM을 통해 분석된 의도에 따라 경로 생성 및 관제 웹 중계가 이루어집니다.

---

## 📊 5. 데이터셋 및 학습 (Data)
* **자체 데이터 구축:** 실제 보행 환경에서 시각장애인의 이동을 방해하는 주요 객체 중심의 데이터셋(2,961개 객체)을 구축했습니다.
* **데이터 증강:** 다양한 조명(밝기 ±15%) 및 촬영 조건(회전, 노출)에 대응하기 위해 데이터를 증강하여 총 1,510장의 이미지를 학습에 활용했습니다.
* **최적 모델 선정:** mAP50 및 추론 속도(Pure FPS)를 종합 고려하여 모바일 환경에 최적화된 모델을 최종 선택했습니다.
